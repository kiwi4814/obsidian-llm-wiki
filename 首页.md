# 首页

这是我的 Obsidian LLM Wiki，由我和 Hermes Agent 共同维护。

## 常用入口

- [[热缓存]]
- [[20-知识库/索引]]
- [[操作日志]]
- [[术语表]]

## 主要区域

- `00-收集箱/`：原始输入
- `10-每日记录/`：时间线
- `20-知识库/`：长期知识
- `30-输出/`：可复用成果
- `40-项目/`：项目资料
- `50-资源/`：模板、Prompt、清单、参考资料
- `90-归档/`：历史归档
- `附件/`：图片、PDF、音频等资源

## 当前重点

- 搭建 Hermes + Telegram + Obsidian 的 LLM Wiki 系统
- 建立稳定的中文知识库规范
- 让 Telegram 输入可以自动捕获、整理、提交

## 维护规则

- 原始输入不删除
- 重要结论要有来源
- 查询结果有价值就回写
- 每次维护后更新索引、热缓存和操作日志
