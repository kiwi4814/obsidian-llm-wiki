# PT 站点自动 RSS 下载 + 自动整理

## 目标

实现 PT 站点 RSS 种子的自动下载、限速、整理、备份和清理全链路自动化。

## 链路

```mermaid
flowchart LR
    RSS[RSS 订阅] --> DB[缓存到数据库<br/>标记为未下载]
    DB --> CHECK{磁盘空间足够?}
    CHECK -->|是| ADD[qB 添加任务<br/>设路径/限速/分类]
    ADD --> SEED[做种 ≥1天]
    SEED --> MOVE[转移目录]
    MOVE -->[[网盘自动备份]]
    MOVE --> WAIT[等待24h]
    WAIT --> DEL[删除种子和文件]
```

## 功能模块

| 模块 | 说明 | 可配置 | 可选启用 |
|------|------|--------|---------|
| RSS 缓存 | 订阅 RSS，缓存种子信息到数据库 | ✅ | ✅ |
| 磁盘检测 | 按序检查空间，排队添加 | ✅ | ✅ |
| 限速与分类 | 设置每种子速度限制和类别 | ✅ | ✅ |
| 做种管理 | 达到做种时间后转移 | ✅ | ✅ |
| 文件整理 | 硬链接重命名，大模型辅助处理标题 | ✅ | ✅ |
| 钩子通知 | 通知网盘备份完成 | ✅ | ✅ |
| 定时清理 | 超过时间后删除种子和文件 | ✅ | ✅ |
| 可视化 | 种子状态流程看板 | ❌ | ✅ |
