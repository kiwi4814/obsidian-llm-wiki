# Codex 推荐的 API 中转服务

Codex 推荐 Right 和 Packy（CC 太贵了也用不到），老牌便宜大碗，掺水率很低（自己用了几天感觉）。

## 中转原理

中转本身是从 AI 供应商算力池子里黑的 token，可以做到价格很便宜。

## 掺水问题

掺水与否和价格并不是直接相关的，而是和中转站有没有良心相关：

- 中转站想多捞一份，就偶尔掺点廉价模型进去
- 更过分的是从头到尾都给你便宜模型用
- 轻度用户也感觉不出来

**点名批评**：咸鱼野生中转
