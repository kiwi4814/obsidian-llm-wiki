2. microsoft/markitdown — 微软出品，文档转 Markdown 神器
⭐ 131,296 Stars | Python

这个项目我已经用了一个多月了，说真的，太好用了。
以前写文章要引用 PDF 或者 Word 里的内容，要么截图，要么手动排版。现在直接丢给它，几秒钟就把 PDF、Word、Excel、PPT、HTML 全转成干净的 Markdown，格式保持得相当不错。
好用在哪：
支持 PDF/Word/Excel/PPT/HTML/图片/音频/邮件 等十几种格式
转换后格式不乱，表格、列表基本保持原样
微软亲儿子，更新频率高，bug 修得快
适合谁用：需要经常处理文档的、写文章要引用资料的、做知识整理的。
怎么获取：GitHub 搜 microsoft/markitdown，pip install 一下就能用。或者直接 pip install markitdown。
