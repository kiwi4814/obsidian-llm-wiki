# 文章/故事转漫画脚本 AI 提示词

## 适用场景

将科学论文、人物传记、历史事件、知识科普文章等转化为 **知识传记漫画（Knowledge Comic）** 的结构化脚本。适用于 LLM / AI 绘图工具。

## 效果参考

- 案例：http://t.cn/AX4KQ2cA
- 脚本示例对话：http://t.cn/AX4KQ2c2
- 画图示例对话：http://t.cn/AX4KQ2cL

## 提示词模板

```
请为一本若干页（另加1页封面）的原创知识传记漫画创作详细的结构和情节设计。
本漫画采用《Logicomix》式的叙事风格，适合呈现科学探索历程、跨越数十年的时间线，
以及复杂概念的可视化解释。

### 视觉风格定义

- **线条清晰（Ligne Claire）**：参考 Hergé《丁丁历险记》或 Joost Swarte 的画风
  ——轮廓线均匀、干净利落；背景写实细致，人物略带卡通化但五官特征鲜明、易于辨认。
- **色彩分层叙事**：运用明快的色块区分不同时空场景。
  早期年代采用泛黄的复古暖色调；现代科技场景使用冷峻的蓝灰色；
  关键突破时刻以高饱和的亮色强调。
- **打破第四面墙**：设置一位或多位"旁白者"角色（如晚年的主人公或核心见证者），
  他们可以穿梭于画面之间，直接面向读者进行解说、评论或提问，形成对话式叙事。

### 要求

**1. 封面设计：**
- 设计一个兼具学术厚重感与视觉吸引力的封面
- 标题字体应体现知识/科学主题
- 封面构图需暗示核心主题，可包含关键人物剪影、标志性符号或概念图示
- 明确标注副标题或时间跨度（如"1986-2012"）以建立史诗感

**2. 清晰布局：**
- 逐页详细描述封面及每一页的画面布局
- 说明每页的分镜数量与构图安排，建议每页3-6个分镜
- 区分"主叙事层"与"旁白层"的视觉处理方式
- 为复杂概念预留"概念图解分镜"
- 标注每个分镜的大致比例（全页、半页、1/3页、小格等）
- 页面比例为 --ar 2:3

**3. 生动细致：**
- 以丰富且具体的语言描述每个分镜的视觉细节
- 包括：角色姿态与表情、场景环境与时代特征、光影氛围、镜头角度
- 对于科学概念可视化，需描述如何将抽象概念转化为具象画面
- 注意不同时空的视觉区分

**4. 叙事结构：**
- 以"封面"及"第1页"至"第N页"清晰区分漫画的不同部分
- 建议采用非线性叙事
- 每页末尾可设置"钩子"以推动翻页欲望
- 在关键情节节点安排"定格时刻"

**5. 对话与文字设计：**
- 对话框风格：主叙事层使用标准椭圆对话框；旁白者使用方形或手写风格文字框
- 科学术语首次出现时可用特殊标注
- 文字精炼有力，避免大段说教
- 所有对白使用中文

**特别注意：**
- 本阶段仅需提供详细、清晰、富有想象力的页面布局和情节描述
- 每个分镜描述应包含足够的视觉信息，使其可独立作为 AI 绘图指令使用
```
