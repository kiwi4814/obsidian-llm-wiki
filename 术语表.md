# 术语表

## Hermes Agent

运行在本地或 VPS 上的自主 AI agent。这里主要用它维护 Obsidian vault。

## Obsidian vault

Obsidian 的笔记文件夹，也是 Hermes 读写的知识库根目录。

## LLM Wiki

由大模型持续维护的 Markdown 知识库。核心不是一次性问答，而是长期编译和维护知识。

## frontmatter

Markdown 文件顶部的 YAML 元数据区域。

## wikilink / 双向链接

Obsidian 使用的 `[[笔记名]]` 链接语法。

## 热缓存

保存当前最重要上下文的小型动态索引文件，用于减少 Hermes 每次重新扫描全库的成本。
