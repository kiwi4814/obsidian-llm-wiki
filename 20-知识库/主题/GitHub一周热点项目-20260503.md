# GitHub 一周热点项目（2026.04.27～2026.05.03）

## 主题概述

本周 GitHub 热点项目集中在 AI 工具和开发者技能生态领域：

- **AI 终端工具**：Warp 将 AI 集成到终端体验
- **安全工具**：Hackingtool 提供一站式渗透测试工具箱
- **AI 编程技能**：Skills 和 awesome-codex-skills 展现 Claude Code/Codex 技能生态的蓬勃发展
- **AI 视频生成**：Pixelle-Video 实现全自动短视频制作

## 核心问题

1. AI 如何改变开发者工具链？
2. Claude Code 和 Codex 的技能生态如何发展？
3. AI 视频生成工具的自动化程度如何？

## 当前结论

- AI 终端和 AI 编程助手正在深度融合开发者工作流
- Claude Code/Codex 技能包成为社区热点，TypeScript 生态占主导
- AI 视频工具开始实现"全自动化"，降低内容创作门槛

## 子主题

### 1. Warp – AI 终端工具

**GitHub**：https://github.com/warpdotdev/warp

**简介**：将 AI 能力集成到终端的现代化工具，支持智能命令补全、错误解释和自然语言交互。

**关键特征**：
- AI 驱动的命令建议
- 现代化 UI 体验
- 团队协作功能

---

### 2. Hackingtool – 一站式黑客工具箱

**GitHub**：https://github.com/Z4nzu/hackingtool

**简介**：集成多种渗透测试和安全审计工具的集合，适合安全研究人员和红队使用。

**关键特征**：
- 工具分类清晰
- 一键安装部署
- 涵盖信息收集、漏洞利用、密码破解等

---

### 3. Skills – TypeScript 顶流的 Claude Code 技能包

**GitHub**：https://github.com/mattpocock/skills

**简介**：Matt Pocock 维护的 Claude Code 技能集合，以 TypeScript 为主，提供高质量的编程辅助技能。

**关键特征**：
- TypeScript 生态专注
- 社区认可度高
- 实用编程场景覆盖

---

### 4. awesome-codex-skills – Codex 技能生态精选

**GitHub**：https://github.com/ComposioHQ/awesome-codex-skills

**简介**：OpenAI Codex 技能的精选列表，收集社区贡献的各类 Codex 技能包。

**关键特征**：
- 生态聚合
- 分类整理
- 社区驱动

---

### 5. Pixelle-Video – AI 全自动短视频

**GitHub**：https://github.com/AIDC-AI/Pixelle-Video

**简介**：AI 驱动的全自动短视频生成工具，从脚本到成片实现端到端自动化。

**关键特征**：
- 全流程自动化
- AI 生成脚本、画面、配音
- 降低视频创作门槛
